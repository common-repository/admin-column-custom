== Description ==

Customise columns on the administration screens for posts, pages and users.

= The plugin allows using =

* Manage posts columns.
* Manage pages columns.
* Manage users columns.

[Documentation](http://photoboxone.com/category/documents/)


== Installation == 
#### Thank you for your interest in the Admin Column plugin.
### Minimum requirements.    

* Wordpress 3.0+ 
* PHP 5.x  
* MySQL 5.x 

This section describes how to install the plugin wordpress and get it working. 
e.g.

1. Upload folder 'plugin-name' to the `/wp-content/plugins/` directory. 
2. Activate the "Admin Column" plugin through the plugins menu in the WordPress. 
3. Go to "Admin Column" in Setting.

If any problem occurs, please read documents Wordpress.
